# Карта гипотез — Claude Skill

**Интерактивный фасилитатор для создания стратегических карт по методологии Александра Бындю**

Этот skill помогает построить Карту гипотез через диалог с Claude, валидирует типовые ошибки и генерирует готовый файл для Excalidraw.

---

## Что такое Карта гипотез?

Карта гипотез — это технология стратегирования, которая визуализирует путь от цели до конкретных задач через гипотезы об изменении поведения субъектов.

```
┌─────────┐     ┌───────────┐     ┌───────────┐     ┌─────────┐
│  ЦЕЛЬ   │ ◄── │ СУБЪЕКТЫ  │ ◄── │ ГИПОТЕЗЫ  │ ◄── │ ЗАДАЧИ  │
│(зелёный)│     │(оранжевый)│     │ (жёлтый)  │     │ (синий) │
└─────────┘     └───────────┘     └───────────┘     └─────────┘
```

### Ключевые понятия

| Элемент | Описание | Пример |
|---------|----------|--------|
| **Цель** | Измеримый результат, к которому стремимся | "Увеличить выручку с 10М до 50М ₽/мес" |
| **Субъект** | Тот, чьё поведение хотим изменить (НЕ исполнитель) | "Клиенты B2B с оборотом >100М" |
| **Гипотеза** | Формализованная идея изменения поведения | "ЕСЛИ... ТО... ПОТОМУ ЧТО... ТОГДА..." |
| **Задача** | Конкретная активность для проверки гипотезы | "Интеграция с API банков" |

### Формат гипотезы

```
ЕСЛИ мы [наше воздействие],
ТО [субъект] [изменит поведение],
ПОТОМУ ЧТО [опора на мотивацию субъекта],
ТОГДА [метрики цели изменятся]
```

## Установка

Скопируйте папку `hypothesis-map` в `.claude/skills/` вашего проекта:

```bash
cp -r hypothesis-map /path/to/your/project/.claude/skills/
```

Запустите Claude Code в вашем проекте — skill будет доступен автоматически.

## Использование

### С Claude Code

**Вариант 1:** Используйте команду:

```
/hypothesis-map
```

**Вариант 2:** Начните диалог:

```
Помоги построить карту гипотез для увеличения продаж
```

Пример ответа Claude:

```
Claude: Давайте создадим Карту гипотез!

        Начнём с цели. По каким критериям в конце работы
        вы поймёте, что достигли успеха?

        Опишите желаемое состояние, к которому стремитесь.
        Не задачу ("внедрить CRM"), а результат ("увеличить продажи").
```

Claude проведёт вас через все этапы:
1. **Цель** — определение результата и метрик
2. **Субъекты** — на кого влияем, их боли и желания
3. **Гипотезы** — формулировка в формате ЕСЛИ-ТО-ПОТОМУ ЧТО-ТОГДА
4. **Задачи** — конкретные активности
5. **Генерация** — создание .excalidraw файла

## Валидация

Skill автоматически проверяет 8 типовых ошибок:

### Ошибки в целях

| Ошибка | Признак | Пример |
|--------|---------|--------|
| Задача вместо цели | Глаголы: "внедрить", "создать" | ❌ "Внедрить CRM" → ✅ "Увеличить конверсию до 15%" |
| Гипотеза вместо метрики | "если", "нужно", "следует" | ❌ "Нужно больше лидов" → ✅ "Лиды: 100 → 500/мес" |

### Ошибки в субъектах

| Ошибка | Признак | Пример |
|--------|---------|--------|
| Исполнитель вместо субъекта | Внутренний отдел/сотрудник | ❌ "Отдел продаж" → ✅ "Клиенты B2B" |
| Наша боль, не субъекта | "мы", "нам", "наш" | ❌ "Нам мало лидов" → ✅ "Сложно сравнить товары" |

### Ошибки в гипотезах

| Часть | Ошибка | Пример |
|-------|--------|--------|
| ЕСЛИ | Действие субъекта, не наше | ❌ "клиент увидит" → ✅ "мы запустим рекламу" |
| ТО | Не о поведении субъекта | ❌ "система быстрее" → ✅ "клиент проводит >5 мин" |
| ПОТОМУ ЧТО | Не связано с мотивацией | ❌ "современный подход" → ✅ "боится потерять деньги" |
| ТОГДА | Не связано с метриками | ❌ "будет хорошо" → ✅ "конверсия вырастет до 8%" |

## Пример из методологии

**Ситуация:** Жители дома хотят, чтобы собачники не выгуливали питомцев у их цоколя.

https://blog.byndyu.ru/2025/12/blog-post.html

### ❌ Плохая формулировка

> Цель: "Сделать так, чтобы собачники выгуливали питомцев подальше"

Это задача, а не цель — описывает действие, а не результат.

### ✅ Правильная карта

**Цель:**
- Сохранить цоколь здания чистым
- Метрика: Загрязнений 10/день → 0
- Балансирующие: Не поругаться с соседями, остаться в рамках закона

**Субъект:** Владельцы собак, гуляющие у крыльца
- Боль: "Боюсь, что собака съест что-то опасное"
- Желание: "Хочу гулять где удобно"

**Гипотеза:**
- **ЕСЛИ** сделать предупреждающую надпись с заботой о том, что приближение к цоколю опасно для собаки,
- **ТО** владельцы начнут обходить наш цоколь,
- **ПОТОМУ ЧТО** они боятся вреда своим питомцам,
- **ТОГДА** количество загрязнений снизится до 0

**Задачи:**
- Придумать текст надписи
- Разместить на видном месте
- Наблюдать за результатом
